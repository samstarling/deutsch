Modal Verbs

A type of verb that is used to indicate modality – that is, likelihood, ability, permission, and obligation.

müssen

To be obliged to, to have to.

  • Ich muss ins Kino gehen.
  • Du musst ein neues Fahrrad kaufen.
  • Er/sie/es muss oben sein.
  • Wir müssen draussen bleiben.
  • Ihr müsst wirklich kalt sein!
  • Sie müssen nach Hause gehen.

dürfen

To be allowed to.

  • Darf ich rauchen?
  • Du darfst ...
  • Er/sie/es darf ...
  • Wir dürfen ...
  • Ihr dürft ...
  • Sie dürfen ...

können

To be able to.

  • Ich kann das Saxophon spielen.
  • Du kannst ...
  • Er/sie/es kann ...
  • Wir können ...
  • Ihr könnt ...
  • Sie können ...

wollen

To want.

  • Ich will ein Pony.
  • Du willst ...
  • Er/sie/es will ...
  • Wir wöllen ...
  • Ihr wollt ...
  • Sie wöllen ...

sollen

Literally: shall. To be expected to.

  • Soll ich zurückgehen?
  • Du sollst ...
  • Er/sie/es soll ...
  • Wir sollen ...
  • Ihr sollt ...
  • Sie sollen ...

mögen

To like.

  • Ich mag käse.
  • Du magst ...
  • Er/sie/es mag ...
  • Wir mögen ...
  • Ihr mögt ...
  • Sie mögen ...

möchten

To wish, to desire.

  • Ich möchte bitte einen Kaffee.
  • Du möchtest ...
  • Er/sie/es möchte ...
  • Wir möchten ...
  • Ihr möchtet ...
  • Sie möchten ...
